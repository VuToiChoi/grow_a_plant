# grow_a_plant
Ett spel där man tar hand om en planta som ska växa.

##Deltagare
Olle Bjäräng:
 - Menyerna och input så att de fungerade.
 - Interface.
 - Main_handler som binder ihop de olika delarna i projektet.
 - Koddokumentation och UML-diagram för dessa delarna.
 - Tutorial

Fabian Carlsson:
- Music och ljudeffekt
- Sound_handler
- Full screen
- Hitta och lägga till font
- En tidigare version av Text_handler
- Generera bilder
- Generell design av prototyp i figma

Huy Vu Dang: 
-   API (Weather_handler)
-   Data_handler
-   Plant_handler
-   Time_handler
-   Hur flödet av prototypen ska vara
-   Spara ner tiden och plantans information lokalt på datorn


Olle Bjäräng och Huy Vu Dang:
- Skapade grundläggande struktur för all kod
- Prototyp dokumentation
- Backlog i google sheet
- Matematiska funktioner för hur plantan, vatten, gödning och hur mycket växten har växt

Alla:
- Trello
- Sprint reviews
- Planering för sprinterna
- Användarmanual (beskrivning)
